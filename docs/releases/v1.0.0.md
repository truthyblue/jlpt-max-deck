# JLPT MAX Deck v1.0.0

배포일: 2026-07-22

v1.0.0은 JLPT MAX Deck의 첫 정식 공개 릴리스입니다. 이 릴리스는 완성 APKG를
나눠 주지 않습니다. 사용자가 정식으로 구한 PDF 17개와 공개 빌더를 자신의
컴퓨터에서 함께 사용해 개인 학습용 APKG를 만듭니다.

## macOS 실제 전체 빌드 검증

v1.0.0 bundle과 정확한 PDF 17개·785쪽을 사용해 macOS에서 전체 빌드를 실제로
수행했습니다.

| 항목 | 결과 |
| --- | --- |
| 컴퓨터 | macOS 26.5.1 arm64 |
| Python / Anki package | 3.13.5 / 25.9.2 |
| PDF | 17개, 785쪽 |
| 검증 상태 | `passed`, `unresolved: 0` |
| APKG 파일 | `JLPT-MAX덱-1.0.0.apkg` |
| APKG 크기 | 836,810,249 bytes |
| macOS APKG SHA-256 | `7add6df097b7d00f92f52d66dc7c43ff76520e37242bbe4178cc8710b99db958` |
| logical APKG SHA-256 | `0ada593a07b5ab5a02ea1261b4fa1d58efb706f49323a02146558a8a1ce7e6ff` |

APKG는 ZIP 내부 정보 때문에 똑같은 덱이어도 운영체제나 실행 시점에 따라 파일
SHA-256이 달라질 수 있습니다. 덱 내용이 같은지 판단할 때는 logical APKG hash를
기준으로 사용합니다. macOS APKG SHA-256은 위 검증 실행의 정확한 파일을
기록한 값입니다.

## Windows 검증 범위

Windows x64에서는 공개 자료로 실행하는 자동 테스트와 PowerShell 스크립트
문법 검사를 통과했습니다. 그러나 Windows x64에서 PDF 17개를 쓴 전체 빌드는
아직 수행하지 않았습니다.

따라서 이 릴리스는 **Windows의 실제 17-PDF 전체 빌드 성공을 입증하는 자료가
아닙니다.** Windows에서 실패하면 검증을 우회하지 말고, PDF나 완성 APKG를
첨부하지 않은 채 오류 메시지와 비민감 report 필드만 제보해 주세요.

## v1.0.0에서 달라진 학습 기능

- 어휘 카드 답 오른쪽 위의 `재생` 메뉴에서 예문 자동재생을 켜고,
  `첫 예문` 또는 `모든 예문`을 고를 수 있습니다.
- 음성을 직접 누르면 남은 자동재생을 멈추고 선택한 음성을 처음부터
  재생합니다.
- 예문 자동재생 선택은 기기마다 따로 저장됩니다.
- JLPT MAX덱 루트 덱 설명과 모든 카드 답 아래에 `덱 안내 · 업데이트`
  링크가 있습니다. 사용자가 누를 때만 프로젝트 페이지를 열며 PDF나
  카드 내용을 전송하지 않습니다.

## 배포하지 않는 파일

아래 파일은 GitHub Release에 올리지 않습니다.

- 출판사 PDF
- 완성 APKG
- PDF 추출물
- `source-proof.json`
- 개인 경로, 계정 정보 또는 개인용 데이터

완성 APKG는 개인 학습용으로만 보관하세요. 업로드, 미러링, 판매, 공유하지
마세요.
