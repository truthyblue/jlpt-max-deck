1 · 설치

앱을 고르고 바로 설치합니다.

현재 접속한 기기의 탭이 자동으로 열립니다. 이름이 비슷한 앱이 많으므로 아래 다운로드 버튼을 이용하세요.

무료 · ANKI DESKTOP

macOS

Apple silicon과 Intel 중 내 Mac의 칩에 맞는 설치 파일을 선택합니다.

  1. Apple 메뉴 → 이 Mac에 관하여에서 ‘칩’ 또는 ‘프로세서’를 확인합니다.
  2. Apple M 계열이면 Apple silicon, Intel이라고 표시되면 Intel 설치 파일을 받습니다.
  3. 다운로드한 DMG를 열고 Anki를 Applications 폴더로 드래그한 뒤 실행합니다.
macOS용 Anki 받기 ↗ 설치 도움말
무료 · ANKI DESKTOP

Windows 10+ x64 · Windows 11 ARM

공식 Anki 앱은 두 환경을 지원합니다. 다만 JLPT MAX 덱 빌더의 Windows 실행 범위는 x64 컴퓨터입니다.

  1. 설정 → 시스템 → 정보에서 ‘시스템 종류’를 확인합니다.
  2. x64 기반이면 Windows x64, Windows 11 ARM이면 Windows ARM 설치 파일을 받습니다.
  3. 다운로드한 MSI를 실행하고 화면 안내에 따라 설치한 뒤 시작 메뉴에서 Anki를 엽니다.
Windows용 Anki 받기 ↗ 설치 도움말
유료 · 공식 iOS 앱

iPhone · iPad

AnkiMobile Flashcards를 설치합니다. App Store의 제공자는 Anki Software, LLC입니다.

  1. 아래 버튼으로 App Store를 엽니다.
  2. 앱 이름과 제공자를 확인하고 구입·설치합니다.
  3. AnkiMobile을 한 번 실행해 덱 목록이 열리는지 확인합니다.
App Store에서 받기 ↗ AnkiMobile 도움말
무료 · 오픈소스 Android 앱

Android

AnkiDroid Flashcards를 설치합니다. 개발자는 AnkiDroid Open Source Team, 패키지는 com.ichi2.anki입니다.

  1. 아래 버튼으로 Google Play를 엽니다.
  2. 개발자와 패키지를 확인하고 설치합니다.
  3. 처음 쓰는 경우 Get started를 눌러 덱 목록을 엽니다.
Google Play에서 받기 ↗ AnkiDroid 도움말
AnkiApp, Anki Pro는 다른 앱입니다.

이 가이드의 대상이 아닙니다. iPhone/iPad는 AnkiMobile, Android는 AnkiDroid인지 확인하세요.

2 · 첫 실행

덱 목록이 열리면 됩니다.

표시 언어와 초기 안내를 확인합니다. 아직 JLPT MAX덱이 보이지 않는 것이 정상입니다.

  1. 처음이면 기본 프로필로 시작합니다.

    기존 컬렉션과 분리해야 할 때는 Desktop의 ‘파일 → 프로필 전환’ 또는 AnkiMobile의 ‘설정 → Profiles’에서 별도 프로필을 추가할 수 있습니다.

  2. AnkiWeb 계정은 지금 없어도 됩니다.

    APKG를 로컬로 가져오고 공부하는 데 계정은 필수가 아닙니다. 여러 기기의 학습 상태를 이어갈 때 무료 계정을 연결합니다.

  3. 이미 쓰던 자료가 있으면 먼저 백업합니다.

    Desktop과 AnkiMobile은 별도 프로필을 사용할 수 있습니다. AnkiDroid는 가져오기 전에 기존 컬렉션을 백업하세요.

3 · 설치 확인

이제 APKG를 가져오면 됩니다.

  • Anki 앱이 오류 없이 실행됨
  • 덱 목록 화면이 표시됨
  • 가져오기 또는 공유로 열기 기능을 사용할 수 있음
  • APKG를 저장할 여유 공간이 있음
Windows ARM 또는 Linux를 사용하나요?

공식 다운로드 페이지에서 지원되는 Anki 앱을 설치해 APKG를 가져올 수 있습니다. 덱은 macOS 또는 Windows x64 컴퓨터에서 만든 뒤 이 기기로 옮기세요. 이 프로젝트의 전체 빌드 지원 범위와 Anki 앱 자체의 지원 범위는 서로 다릅니다.

가져오기 화면과 덱 설정까지 확인하세요.

먼저 가져오기 옵션 권장값에서 학습 진행 상태, 덱 프리셋과 노트 타입 병합을 맞춥니다. 가져온 뒤에는 기기별 덱 옵션과 FSRS 설정 안내에서 새 카드 수, 목표 기억률과 답변 버튼 사용법을 확인하세요.

Anki가 열렸다면 덱을 만듭니다.

가이드 안에서 현재 컴퓨터에 맞는 빌드 명령을 복사하고, 완성 APKG를 공부할 기기로 바로 가져올 수 있습니다.
